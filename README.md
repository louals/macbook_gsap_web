# MacBook 3D Viewer

A **3D interactive MacBook model** built with **Three.js**, animated with **GSAP**, styled with **Tailwind CSS**, and powered by **React**.

Experience smooth 3D transitions, realistic model rendering, and a modern responsive UI.
